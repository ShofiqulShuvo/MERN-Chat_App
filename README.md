# MERN Chat App

## installation

<p> open two terminal in your root directory</p>

##### First Terminal

```
    cd frontend
    npm install
```

##### Second Terminal

```
    cd backend
    npm install
```

## Commands for run the app

##### First Terminal

```
    npm start
```

##### Second Terminal

```
    npm start
```
